## Project Overview

ESO FontChanger is an Elder Scrolls Online (ESO) addon written in Lua. It replaces in-game fonts (UI, chat, books, nameplates, SCT) with custom fonts, configurable via an in-game settings menu. Fork of the original FontChanger by Ferrety/Antisenil.

**Depends on:** LibAddonMenu-2.0 (LAM2) for the settings panel.
**ESO API Version:** 101049
**Saved Variables:** `FontChangerSettings`

## Architecture

The addon loads files in this order (defined in `FontChanger.txt`):

1. **Constants.lua** — Declares the global `FontChanger` table (`FC`). Defines all dropdown option arrays (font choices/values, styles, weights, sizes, scales) and the `FC.defaults` table with default saved variable values.
2. **CustomFontOptions.lua** — **Auto-generated by `fonts/slug.bat` or `fonts/slug.sh`; do not edit directly.** Populates `FC.CUSTOM_FONT_CHOICES` and `FC.CUSTOM_FONT_VALUES` from slug files in `fonts/slugs/`.
3. **Menu.lua** — Builds the LAM2 settings panel (`/fc` slash command). Uses data-driven helper functions (`dropdown()`, `slider()`) to generate controls from saved-variable keys. Merges default + custom font lists into dropdown options. Each setting's `setFunc` calls the relevant `FC:Set*` method for live preview.
4. **FontChanger.lua** — Core logic. `FC:Initialize()` loads saved variables, captures vanilla nameplate/SCT settings, registers events, and applies all fonts. Key methods:
   - `SetUIFonts()` — iterates all global `Zo*` userdata objects, pattern-matches font paths, and replaces them; applies `menu_style` when set
   - `SetNameplateFont()` / `SetSCTFont()` — uses ESO API for nameplate/SCT fonts, with keyboard/gamepad mode branching
   - `ChangeChatFonts()` — sets chat entry box, chat window, and forces existing chat buffers to pick up new fonts
   - `HookLoreReader()` — defers a `ZO_PostHook` on `LORE_READER:ApplyMedium` to `EVENT_PLAYER_ACTIVATED`, replacing book/letter/tablet body and title fonts
   - `RestoreVanillaFonts()` — resets nameplate and SCT fonts to the vanilla settings captured at first initialization
   - `SetupEvents(toggle)` — registers/unregisters `EVENT_PLAYER_ACTIVATED`, `EVENT_ZONE_CHANGED`, `EVENT_GAMEPAD_PREFERRED_MODE_CHANGED`
5. **Compatibility.lua** — PerfectPixel addon integration. Uses `ZO_PreHook` on `PP.Core` to inject FC's fonts before PP initializes (handles PP loading before FC). Uses `ZO_PostHook` on `PP.compass` and `PP.keybindStrip` to reapply FC scaling to compass cardinal directions, boss bar text, and keybind strip fonts.

## Adding Custom Fonts

Use the conversion script for your platform (`slugfont.exe` from your ESO install is required):
- **Windows:** `fonts/slug.bat`
- **Linux:** `fonts/slug.sh` (requires Wine)

Both scripts:
1. Convert all `.ttf` files in `fonts/ttfs/` to `.slug` files in `fonts/slugs/`
2. Regenerate `CustomFontOptions.lua` with the new font list

Only files prefixed with `FC` in `fonts/slugs/` and `fonts/ttfs/` are tracked by git (see `.gitignore`).

## Key Patterns

- All state lives on the `FontChanger` global table (aliased as `FC` locally in each file)
- Saved variables use a flat structure: `FC.SV.menu_font`, `FC.SV.nameplate_size`, etc.
- `FC.defaults` uses the same keys as saved variables (e.g., `FC.defaults.menu_font` matches `FC.SV.menu_font`)
- Font paths use ESO's `AddOnName/path/to/font.slug` format (e.g., `FontChanger/fonts/slugs/FCUI.slug`)
- Gamepad mode support is toggled via `FC.SV.gamepad_fonts_enabled` and checked with `IsInGamepadPreferredMode()`
